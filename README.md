Ultimate is a feature-packed industrial noise generator and audio workstation built in Streamlit. Adjust signal flow, dial in BPM, select scales, layer synthesizer waveforms, chain effects in any order, generate variations, and manage your session — then download everything.

## ✨ Features

- **Signal flow customization** — reroute effects and components in any order
- **BPM control** — set beats per minute for rhythm-based noise
- **Note & scale selection** — choose root notes and scales for the synthesizer
- **Built-in synthesizer** — multiple waveforms (sine, saw, square, noise)
- **Advanced effects chain** — reverb, delay, distortion, compression, and more
- **Variations generator** — auto-generate variations of each sample
- **Interactive waveform editor** — zoom, pan, and inspect your audio
- **Session management** — store and download all generated sounds

## 🚀 Quick Start

```bash
git clone https://github.com/RhythrosaLabs/ultimate.git
cd ultimate
pip install -r requirements.txt
streamlit run app.py
```

Or use the [live app](https://ultimate.streamlit.app).

## 🛠️ Tech Stack

- **Python / Streamlit** — app framework
- **Pedalboard** — audio effects (Spotify)
- **NumPy** — signal generation
- **soundfile** — audio I/O
